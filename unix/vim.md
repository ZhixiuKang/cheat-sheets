# vim

## File IO

### opening files

* [`<c-p>`](https://github.com/kien/ctrlp.vim#basic-usage) - fuzzy finder
 - `<c-f>` - go to next mode
 - `<c-v>` - open in a vertical split
 - `<c-v>` - open in an horizontal split
 - `<c-z>` - mark/unmark multiple files
  
### buffers

* `ls` - list all buffers
* `:b <tab>` - go to buffer
* `:bd` or `<space> q` - delete buffer

## Editing

### folding

`za` - toggle folding
